# OGSBL_GH
An off-grid DOA estimation method based on Generalized Hyperbolic (GH) prior

The official code for "Jiao J, Wang J, Pan X. Robust DOA Estimation Through Flexible Sparsity-Aware Modeling in Low SNR[C]//OCEANS 2025 Brest. IEEE, 2025: 1-5." and "Direction-of-arrival estimation for weak underwater targets via flexible sparsity-aware modeling".

If there are some questions or errors, please contact jianbojiao@zju.edu.cn
